## 🌍 Projects

Hidden Gemz

HiddenGemz is a full-stack community-driven platform designed to help you discover, share, and save the best local spots. Whether it's a quiet cafe, a bustling bar, or a hidden restaurant, HiddenGemz connects you with authentic local experiences. The project is built using React (v18) with TypeScript and Vite for fast development and efficient bundling. It uses Tailwind CSS for responsive styling, along with TanStack Query for data fetching and caching, React Router for navigation, and React Hot Toast for notifications. On the backend, it’s powered by Node.js and Express.js with Knex.js as the query builder, using SQLite3 in development and PostgreSQL in production. Authentication is handled securely through Auth0 with JWT-based authentication.

knex-todo-cli

This is a simple command-line todo application built using Knex as the query builder to interact with a SQLite3 database. It allows users to manage a list of todos directly from the terminal, with all data being stored persistently in a database rather than in memory. This project focuses on building a strong understanding of backend data handling, including creating, reading, updating, and deleting records using Knex.

my-fullstack-collection

This project was inspired by my love for FC Barcelona. It focuses on a player list that displays a players number when clicked on. It contains node modules and folders for databases, routes, API requests on the server side, along with React components on the client side. It also integrates React Query to manage data fetching and state efficiently, helping streamline communication between the frontend and backend.
